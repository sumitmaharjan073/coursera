Representation of number types on number line

The number types are represented on number line using the below table:

Number Type

Range of numbers

Number Line

Natural Numbers

1 to ∞

  

Whole Numbers

0 to ∞

  

Integers

-∞ to ∞

  

Rational Numbers

-∞ to ∞

  

 

While representing rational numbers on number line,

  • Between two integers, the line should be divided into number of equal parts which is same as the denominator of the rational number. For eg, to represent 1/3, three equal parts are made from 0 to 1 representing 1/3, 2/3 and 3/3 (that is 1) respectively.
